# 🏥 Epic EHR + MyChart Healthcare Analytics Dashboard

## 📌 Overview

This project demonstrates how healthcare organizations can leverage Epic EHR and MyChart data to drive operational efficiency, patient engagement, and population health initiatives using Power BI, Python, and SQL.

The solution simulates real-world healthcare analytics workflows commonly used in enterprise healthcare systems such as Kaiser Permanente.

---

# 🎯 Business Problem

Healthcare systems generate massive amounts of operational and patient engagement data, but actionable insights are often fragmented across platforms.

This project aims to:
- Reduce patient no-shows
- Improve MyChart adoption
- Optimize telehealth utilization
- Support population health management
- Deliver executive-level healthcare insights

# 📈 Key Insights

- Patients not using MyChart showed higher no-show rates
- Telehealth appointments reduced missed visits
- Chronic care patients with missed follow-ups represented high-risk cohorts
- Younger patient groups demonstrated lower engagement consistency
